<h3>Mac Productivity</h3>
<p>I prefer to stick with Apple's own apps since (a) they're already included, and (b) they work across all my devices. However, I do have some other apps I like to use:</p>
<ul>
<li><p><a href="https://www.office.com">Microsoft 365</a> - I get a discount on this through my employer, and I'm a <em>huge</em> fan of Excel's Power Query.</p>
</li>
<li><p><a href="https://www.macpaw.com">CleanMyMac</a> - For that 'squeaky clean' feeling on my MacBook Air.</p>
</li>
<li><p><a href="https://www.dropoverapp.com">DropOver</a> - Makes it so much easier to copy and move files around macOS in bulk. </p>
</li>
<li><p><a href="https://www.docker.com">Docker</a> so I can have containerized applications (like a Microsoft SQL Server&mdash;yes it can run on Apple Silicon in an AMD64 linux container thanks to Rosetta 2) when I need it.</p>
</li>
<li><p><a href="https://www.wolfram.com/engine">Wolfram Engine</a> - Quite possibly the most powerful and sophisticated computation system in existence, which drives the well-known <em>Mathematica</em> but does <em>so</em> much more. The engine itself is free for personal and noncommercial use and there is an extension that will let me use Jupyter or Visual Studio Code as the front end for it. Granted, it has a rather steep learning curve, but once I got the hang of it, it's been able do accomplish amazing things.</p>
</li>
</ul>

</ul>

<h2>Musical Instruments</h2>
<ul>
<li><p><a href="https://www.yamaha.com">Yamaha PSR-EW400 Portable Music Workstation</a>. This has a USB host port that carries both MIDI and digital audio, so connecting this to my iPad Pro running GarageBand provides some amazing sonic layers and routes all the audio back through the keyboard's outputs.  Also a second USB host port for external storage.</p>
</li>
<li><p><a href="https://usa.yamaha.com/products/musical_instruments/guitars_basses/silent_guitar/slg200_series/index.html">Yamaha Silent Guitar SLG-200S</a> - Because I live in an apartment and my lack of skills are enough to disturb the neighbors should I use a full acoustic.  The 200S is the steel-stringed version; the 200N is nylon-stringed.</p>
</li>
<li><p>Ibanez V200-BK - That said, I do have a full acoustic guitar, which I bought long before learning about the SLG series. This is an entry-level model and it needs to have the nut adjusted so as not to nearly sever my fingertips at the head of the fretboard when trying to finger a barre chord (barres are the bane of my musical existence).</p>
</li>
<li><p><a href="https://carryonplaying.com">CarryOn 88-key folding digital piano</a> - I admit I bought it because of the novelty. It sounds awful and the keys are shorter and wider with no dynamics or aftertouch control, which poses playing issues when you're used to taller, full-sized keys with dynamics and aftertouch. It does make for a useful (and cheap) 88-key MIDI controller, though, and it has the storied Blackstar name behind it.</p>
</li>
</ul>

</li>
<li><p>Record players (ol' fashioned vinyl) - I have an <a href="https://www.audio-technica.com/en-us/turntables/all/at-lp60xbt-usb">Audio Technica AT-LP60BT</a> for regular listening and a <a href="https://www.numark.com/product/pt01usb">Numark PT01 USB</a> "vinyl-archiving turntable" for digitizing audio LPs and portable use. The Numark is one of the few that offer 78 RPM playback for those old ten-inch shellac albums from the 1920s and '30s (I have a collection of the Dorsey Brothers from that era), and it can run on four "D" batteries for portable use as well.</p>
</li>
</ul>
