This is a static site that is generated from source Markdown, HTML, and Liquid markup using the [Jekyll](https://jekyllrb.com) site engine and hosted on [GitHub Pages](https://pages.github.com). [Cloudflare](https://www.cloudflare.com) provides caching and [DDoS](https://en.wikipedia.org/wiki/Denial-of-service_attack) attack protection.

The style font is "[Computer Modern Serif](https://en.wikipedia.org/wiki/Computer_Modern)," which science, math, and older geeks will recognize as the standard typeface used in $$\rm\TeX$$. [MathJax](https://mathjax.org) provides a library to render $$\rm\TeX$$ and $$\rm\LaTeX$$ strings into something presentable on these pages, and using the Computer Modern Serif typeface as the style therefore keeps things consistent.

The site's source is maintained using Microsoft [Visual Studio Code](https://code.visualstudio.com) for Mac. 

To generate data-driven graphics, I generally use the [Wolfram Language](https://www.wolfram.com) but also make forays into [Pandas](https://pandas.pydata.org) and [Matplotlib](https://matplotlib.org) in [Python](https://python.org). I'm also versed with [Google Apps Script](https://script.google.com), AppleScript, and JavaScript/TypeScript.
